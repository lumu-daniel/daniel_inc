1. Qn: In jQuery terminology, what is the difference between the “jQuery function” and the “jQuery object”?

Ans: A jquery function is depicted by $ and is used to create jQuery objects and register handlers to be invoked when the DOM is ready.
While a jQuery Object is an object the jQuery function returns.
